# Worklog Review Response Template

Use this output shape:

- `Date`: `<YYYY-MM-DD>`
- `Total logged`: `<Xh Ym>`
- `Gap to 8h`: `<Xh Ym>` or `0`
- `Breakdown`:
  - `<ISSUE-KEY>: <summary> (<duration>)`
- `Actions taken`:
  - `<worklog added on ISSUE-KEY: duration>`
  - `<new issue created: TL-XXX>`
- `Next options`:
  - `Continue suggestions to reach 8h`
  - `Stop for now`
